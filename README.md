# 项目简介 python604


## 技术栈

### 后端

- 语言：Python
- 框架：Django
- 数据库：MySQL 5.7.33

### 前端

- jQuery
- Bootstrap
- HTML
- JSP

### 开发工具

- PyCharm

## 功能模块

- 用户管理：注册、登录、个人资料管理等
- 课程管理：课程列表、课程详情、课程学习等
- 练习与测试：在线练习、在线测试、成绩管理等
- 数据可视化：学习进度、成绩分布、用户活跃度等

## 系统角色

- 学生：注册、学习课程、参与练习与测试、查看学习数据等
- 教师：发布课程、管理课程、批改作业、分析学生学习情况等
- 管理员：系统设置、用户管理、课程管理等

## 运行环境

- 操作系统：Linux、Windows、Mac OS
- Python版本：3.6+
- MySQL版本：5.7.33

## 部署步骤

1. 安装Python 3.6+环境
2. 安装MySQL 5.7.33数据库
3. 克隆项目代码到本地
4. 安装项目依赖
5. 配置数据库连接
6. 运行项目

## 目录结构

```
项目名称
├── app            # 应用目录
│   ├── models.py  # 数据库模型
│   ├── views.py   # 视图处理
│   ├── forms.py   # 表单处理
│   └── admin.py   # 管理后台
├── static         # 静态文件目录
├── templates      # 模板文件目录
├── manage.py      # 项目管理脚本
└── requirements.txt # 依赖列表
```

## 核心亮点

- 使用Django框架，具有良好的可扩展性和易用性
- 基于Bootstrap的前端设计，界面美观、响应式布局
- 数据可视化功能，直观展示学习进度与成果
- 支持多角色使用，满足不同用户需求
- 跨平台运行，满足不同操作系统用户的需求
